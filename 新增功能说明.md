# 简易计算器 - 新增功能说明

## 项目概述

基于原有的简易计算器项目，我们新增了多项强大的功能，将简单的四则运算计算器升级为一个功能全面的科学计算器套件。新增功能包括科学计算、历史记录、单位转换等，提供了更丰富的数学计算和用户体验。

## 新增功能概览

### 1. 🧪 科学计算器
### 2. 📜 计算历史记录
### 3. 📏 单位转换器
### 4. 🔢 增强表达式计算

---

## 详细功能说明

### 1. 🧪 科学计算器功能

#### 核心功能
- **三角函数计算**：sin、cos、tan、asin、acos、atan
- **对数函数**：自然对数（ln）、常用对数（log）
- **指数函数**：e^x、平方（x²）、立方（x³）、开方（√）、立方根（³√）
- **其他数学函数**：阶乘（n!）、绝对值（|x|）、倒数（1/x）
- **数学常数**：π（圆周率）、e（自然常数）
- **括号支持**：支持复杂表达式的括号运算
- **负数计算**：支持负数运算（如 8×-7）
- **实时计算**：输入时实时显示计算结果

#### 技术实现
- **ScientificCalculatorUtil.ets**：科学计算工具类，提供所有数学函数的实现
- **科学计算页面**：专门设计的科学计算器界面，包含高级数学函数按钮
- **角度/弧度模式切换**：支持角度（DEG）和弧度（RAD）模式切换
- **UI优化**：参考简易计算器风格，统一按钮样式和布局

#### 使用示例
```typescript
// 正弦函数计算
const sinValue = ScientificCalculatorUtil.sin("30"); // 返回 0.5

// 对数计算
const logValue = ScientificCalculatorUtil.log("100"); // 返回 2

// 幂运算
const powerValue = ScientificCalculatorUtil.power("2", "3"); // 返回 8

// 平方根计算
const sqrtValue = ScientificCalculatorUtil.sqrt("16"); // 返回 4
```

### 2. 📜 计算历史记录功能

#### 核心功能
- **本地持久化存储**：使用HarmonyOS Preferences API存储计算历史
- **历史记录管理**：支持添加、查询、删除、搜索历史记录
- **智能搜索**：支持按表达式或结果关键词搜索
- **时间戳记录**：每条记录包含计算时间
- **滑动删除**：支持滑动删除单条历史记录
- **批量清空**：一键清空所有历史记录

#### 技术实现
- **HistoryManagerUtil.ets**：历史记录管理工具类
- **Preferences存储**：使用@ohos.data.preferences进行本地数据持久化
- **历史记录页面**：专门的历史记录查看和管理页面
- **数据结构**：
  ```typescript
  interface HistoryRecord {
    expression: string;
    result: string;
    timestamp: number;
    id: string;
  }
  ```

#### 使用示例
```typescript
// 添加历史记录
await HistoryManagerUtil.addHistory("2 + 3", "5");

// 获取历史记录列表
const historyList = await HistoryManagerUtil.getHistoryList();

// 搜索历史记录
const searchResults = await HistoryManagerUtil.searchHistory("sin");

// 删除历史记录
await HistoryManagerUtil.deleteHistory(recordId);

// 清空所有历史
await HistoryManagerUtil.clearAllHistory();
```

### 3. 📏 单位转换器功能

#### 支持的单位类别
1. **长度单位**：毫米(mm)、厘米(cm)、米(m)、千米(km)、英寸(in)、英尺(ft)、码(yd)、英里(mi)
2. **重量单位**：毫克(mg)、克(g)、千克(kg)、吨(t)、盎司(oz)、磅(lb)
3. **温度单位**：摄氏度(°C)、华氏度(°F)、开尔文(K)
4. **面积单位**：平方毫米(mm²)、平方厘米(cm²)、平方米(m²)、平方千米(km²)、公顷(ha)、英亩(ac)
5. **体积单位**：毫升(mL)、升(L)、立方米(m³)、加仑(gal)、夸脱(qt)
6. **速度单位**：米/秒(m/s)、千米/时(km/h)、英里/时(mph)、节(kn)
7. **时间单位**：毫秒(ms)、秒(s)、分钟(min)、小时(h)、天(d)、周(w)

#### 核心功能
- **实时转换**：输入值实时计算转换结果
- **单位互换**：一键交换源单位和目标单位
- **常用转换**：提供常用单位转换快捷按钮
- **科学计数法**：自动对大数字和小数字使用科学计数法
- **精确计算**：支持高精度计算，保留6位小数

#### 技术实现
- **UnitConverterUtil.ets**：单位转换工具类
- **温度特殊处理**：温度转换需要特殊算法（非线性转换）
- **选择器组件**：使用HarmonyOS Select组件实现单位选择
- **数据结构**：
  ```typescript
  interface UnitItem {
    name: string;
    symbol: string;
    toBaseFactor: number;
  }

  interface ConversionResult {
    value: number;
    fromUnit: string;
    toUnit: string;
    formatted: string;
  }
  ```

#### 使用示例
```typescript
// 长度转换：米转千米
const result = UnitConverterUtil.convert(1000, 'm', 'km', UnitCategory.LENGTH);
// result = { value: 1, fromUnit: 'm', toUnit: 'km', formatted: '1' }

// 温度转换：摄氏度转华氏度
const tempResult = UnitConverterUtil.convert(25, 'c', 'f', UnitCategory.TEMPERATURE);
// result = { value: 77, fromUnit: 'c', toUnit: 'f', formatted: '77' }

// 获取单位列表
const lengthUnits = UnitConverterUtil.getUnitsByCategory(UnitCategory.LENGTH);
```

### 4. 🔢 增强表达式计算

#### 新增功能
- **括号支持**：支持多层括号嵌套运算
- **复杂表达式**：支持包含函数调用的复杂表达式
- **错误处理**：完善的错误处理和验证机制
- **实时计算**：输入时实时显示计算结果
- **科学计数法**：自动对大数字和小数字使用科学计数法
- **负数支持**：支持负数运算（如 8×-7 = -56）

#### 支持的表达式格式
```
基础运算：2 + 3 * 4
括号运算：(2 + 3) * 4
复杂运算：sin(30) + cos(45)
混合运算：sqrt(16) + log(100) * 2
负数运算：8 * -7
```
